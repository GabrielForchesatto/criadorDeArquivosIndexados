Criador de arquivos indexados
Crie um .txt
Coloque um texto nele
Esse texto passará pelos processos de: Normalização, Stemming. Tokenização e remoção de StopWords
Após isso, as palavras do texto serão indexadas a um dicionário
As palavras serão as Keys, e o nome do arquivo .txt em que elas estão será o Value
Se a mesma palavra está em dois arquivos diferentes, ela receberá ambos os .txt como Value, evitando duplicata da mesma palavra



A pasta .idea contém os encoding pro .txt
